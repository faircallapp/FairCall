# Image assets for FairCall website

Place your images in the `assets/` folder with these exact filenames:

## Logo
| Filename   | Description                    |
|-----------|--------------------------------|
| `logo.png` | FairCall logo (shown in nav)   |

## Screenshots
| Filename      | Caption shown on site        |
|---------------|------------------------------|
| `screen-1.jpeg` or `.png` | Classes                 |
| `screen-2.jpeg` or `.png` | One-tap pick           |
| `screen-3.jpeg` or `.png` | Scoring                |
| `screen-4.jpeg` or `.png` | Participation history  |
| `screen-5.jpeg` or `.png` | Session settings       |
| `screen-6.jpeg` or `.png` | Start session          |

**Note:** The HTML currently expects `.jpeg` for screenshots. To use `.png`, replace `screen-X.jpeg` with `screen-X.png` in `index.html` for each screenshot.

If an image is missing, the site shows a placeholder box with the filename.
